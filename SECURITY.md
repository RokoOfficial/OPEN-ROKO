# Security Policy

## Supported Versions

Only the latest version of OPEN-ROKO is supported with security updates.

## Reporting a Vulnerability

If you discover a security vulnerability, please report it directly to the lead developer and creator:

**RokoOfficial**
[https://github.com/RokoOfficial](https://github.com/RokoOfficial)
