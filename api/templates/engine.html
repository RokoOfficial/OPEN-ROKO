O que é o HMP?

HMP (Human Machine Protocol) é uma linguagem declarativa projetada para automação de processos. Com sintaxe simples e legível, permite criar fluxos complexos de forma intuitiva.

Exemplo Rápido

# Variáveis
SET nome TO "Maria"
SET idade TO 25

# Chamada de Tools
CALL math.sum WITH a=10, b=5
SET resultado TO ${last_result["default"]}

# Condicionais
IF ${resultado} > 10 THEN
    CALL log.print WITH message="Maior que 10!"
ENDIF

# Loops
LOOP 3 TIMES
    CALL log.print WITH message="Iteração ${loop_index}"
ENDLOOP
